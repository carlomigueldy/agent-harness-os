# Learnings

Short, searchable, actionable lessons discovered during {{PROJECT_NAME}} sessions. Linked to relevant files or issues where possible.

## Format

Each entry is a single bullet. Keep it concise. Link to context when available.

```md
- **[Topic]** Lesson learned. Actionable takeaway. → `path/to/file` or #issue-number
```

Capture:
- Repeated mistakes and how to avoid them
- Useful debugging findings
- Repo-specific patterns or quirks
- Framework-specific gotchas
- Better commands or flags discovered
- Testing insights
- Performance findings
- Common failure modes
- Skills that should be created
- Workflows that should be improved
- Worktree/env handling lessons
- Agent-team coordination lessons
- Dynamic-workflow lessons
